# Task List Management System (Trello-Inspired)

## Overview

The **Task List Management System** is a simple Trello-inspired web application that allows users to create, organize, and manage tasks across multiple columns such as **To Do**, **In Progress**, and **Done**.
It provides an interactive interface where users can add, edit, delete, and move tasks between lists, helping track work progress effectively.

This project demonstrates core frontend development concepts such as responsive layouts, DOM manipulation, and browser storage.

## Features

- Create new tasks
- Edit existing tasks
- Delete tasks
- Organize tasks into columns (To Do, In Progress, Done)
- Move tasks between columns
- Responsive user interface
- Persistent storage using **LocalStorage** (tasks remain after page refresh)

## Technologies Used

- **HTML5** – Structure of the application
- **CSS3** – Styling and responsive layout
- **JavaScript** – Task management logic and interactivity
- **LocalStorage** – Store tasks in the browser

## How to Run the Project

1. Clone the repository:

```
git clone https://github.com/Shailo2002/sparkIIT_task_list.git
```

2. Navigate to the project folder:

```
cd task-list-management
```

3. Open the project:

- Double-click **index.html**, or
- Open it in a browser.

No additional setup or installation is required.

## Usage

1. Add a new task to a column.
2. Update or edit tasks as needed.
3. Move tasks between **To Do**, **In Progress**, and **Done** as progress changes.
4. Delete tasks that are completed or no longer required.

All tasks are automatically saved in the browser using **LocalStorage**.
